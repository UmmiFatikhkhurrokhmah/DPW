AJAX & jQuery

Penjelasan dan Contoh Penggunaan

Apa itu AJAX?

AJAX (Asynchronous JavaScript and XML) adalah teknik yang memungkinkan halaman web mengambil atau mengirim data ke server tanpa me-reload seluruh halaman.

Apa itu jQuery?

jQuery adalah library JavaScript yang menyederhanakan manipulasi DOM, event handling, animasi, dan AJAX. Dengan jQuery, kode menjadi lebih singkat dan mudah dibaca.

Contoh jQuery Sederhana:

$("#tombol").click(function() { alert("Halo!"); });
