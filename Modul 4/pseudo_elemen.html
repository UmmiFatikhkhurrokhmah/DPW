Pseudo-elements Selector

1. ::first-line — baris pertama berubah warna

p::first-line { background-color: #f8b4c8; }

Belakangan ini tengah ramai menjadi perbincangan netizen, soal platform chat yang memanfaatkan kecanggihan Artificial Intelligence (AI), yakni Chat GPT. Namun tak sedikit pula dari orang-orang, yang penasaran bagaimana cara pakai Chat GPT.

2. ::first-letter — huruf pertama diperbesar

p::first-letter { font-size: 3em; color: #e75480; }

Bagi yang belum tahu, platform ini mirip seperti chatbot yang dikembangkan oleh Open AI, startup besutan Elon Musk. Artikel ini telah tayang di TribunJatim.com.

3. ::before dan ::after — tambahkan konten sebelum/sesudah

p::before { content: "- " }   p::after { content: " -" }

Ini adalah teks dengan hiasan sebelum dan sesudah paragraf menggunakan pseudo-element

CSS pseudo-element sangat berguna untuk menambahkan dekorasi tanpa mengubah HTML
