Grafis SVG HTML5

Lingkaran (Circle)

Dibuat dengan tag <circle>.
cx, cy = posisi pusat | r = jari-jari
stroke = warna garis tepi | fill = warna isi

Persegi Panjang Biru

Dibuat dengan tag <rect>.
Menggunakan CSS inline: fill, stroke-width, stroke.

Persegi Merah (Sudut Bulat + Transparan)

Atribut rx dan ry untuk sudut membulat.
opacity mengatur transparansi (0 = bening, 1 = penuh).
x, y = posisi pojok kiri atas.

Poligon / Segitiga Hijau

Dibuat dengan tag <polygon>.
Atribut points berisi daftar koordinat titik-titik sudut.
Format: x1,y1 x2,y2 x3,y3 ...
Sangat fleksibel untuk membuat berbagai bentuk segi banyak.

Teks di SVG (Bonus)

Tag <text> menampilkan tulisan sebagai grafis vektor. Bisa diberi warna, font, dan ukuran sesuka hati!

Kesimpulan

SVG (Scalable Vector Graphics) adalah format grafis berbasis XML yang dapat ditampilkan langsung di HTML5. Keunggulannya adalah tidak pecah saat diperbesar karena berbasis vektor. Kita bisa membuat lingkaran (<circle>), persegi panjang (<rect>), poligon (<polygon>), garis (<line>), dan teks (<text>). Referensi: w3schools.com/graphics/svg_intro.asp
